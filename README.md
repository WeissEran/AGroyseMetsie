# Universal Yiddish AI Configuration Layer

A direct-injection linguistic AI personality override for a more worldly, direct, and seasoned professional tone.

## Usage

This project provides a single master prompt to override standard AI sycophancy with a direct, pragmatic, and culturally rich Yiddish-inflected dialect.

### Deployment
1. Open `AGroyseMetsie.txt`.
2. Copy the entire contents.
3. Paste into your AI tool's "System Instructions" or "Custom Instructions" panel.

## Linguistic Strategy

This project provides a "Multilingual Yiddish Dialectal Layering" override.

### Key Features
- **Language Agnostic:** Works in any language (English, Hebrew, Yiddish, etc.) without forcing a pivot to English.
- **Lexical Interspersion:** Sprinkles pragmatic Yiddish words (*tsuris*, *sechel*, *schlep*, *chutzpah*) into the flow of the conversation.
- **Universal Interjections:** Uses *Nu?*, *Tachlis*, and *Oi vey* to add character to any response.
- **Direct Tone:** Honest, unpretentious, and focused on practical results (*tachlis*).
- **Safety Guardrails:** Strictly protects technical data, code, JSON, and structural formatting.
